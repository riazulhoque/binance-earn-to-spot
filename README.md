# binance-earn-to-spot

Helper project to track Binance Airdrop, Launchpool, Earn and Staking balances and guide transfer to Spot Wallet.

---

## 📌 Project Purpose

This repository is created to help understand and manage all Binance earning balances in one place and guide how to move them into the Spot Wallet safely.

This project **does NOT move funds automatically**.
It works as a **tracker, guide, and helper**.

---

## 🔍 What This Project Covers

- ✅ Airdrop rewards
- ✅ Launchpool rewards
- ✅ Flexible Earn balance
- ✅ Locked / Staking balance
- ✅ Total Earn summary
- ✅ Blockchain Earn products
- ✅ Spot Wallet transfer guidance

---

## ⚠️ Important Reality Check

- ❌ This repo cannot directly move assets inside Binance
- ❌ GitHub scripts cannot access your funds without permission
- ✅ Binance controls all transfers
- ✅ This project helps you **track and understand** balances and guides **manual transfer**

---

## 🔁 Earn → Spot Wallet Rules

| Balance Type        | Can Move to Spot? | Notes |
|--------------------|------------------|------|
| Airdrop            | ✅ Yes | Usually auto-added to Spot |
| Launchpool         | ✅ Yes | Auto-distributed |
| Flexible Earn      | ✅ Yes | Redeem required |
| Locked / Staking   | ❌ No  | Wait until lock ends |
| Blockchain Earn    | ⚠️ Depends | Product-based |
